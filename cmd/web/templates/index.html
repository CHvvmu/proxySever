Разработаны функции:

  1. создания пользователя
  2. по ID делает друзей из двух пользователей (curl -X POST -i http://localhost:8800/users/friends -H "content-type: application/json" -d "{\"sourceid\":1,\"targetid\":4}")
  3. по ID удаляет пользователя...........................(curl -X DELETE -i http://localhost:8800/users/delete/4)
  4. по ID обновляет возраст пользователя.........(curl -X PUT -H "content-type: application/json" -d "22" -i http://localhost:8800/users/2)

Просмотр в браузере:

  • всех пользователей (меню)
  • по ID информация о пользователе (кнопка)
  • по ID пользователя всех его друзей (кнопка)
